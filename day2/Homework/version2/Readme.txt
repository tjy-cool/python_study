本程序主要模拟人们的购物的程序
主要流程有：
1. 登陆
    提示输入用户名，用户名存在需要先判断是否已经被锁定了账户（一次性输入三次错误密码），
    如果锁定输出锁定时间，然后直接退出，
    否则提示输入密码，并记录错误密码次数，如果已达三次，则锁定用户并退出程序
    如果用户名不存在则提示是否创建新用户和输入用户密码
2. 首次购物提示输入薪水
    如果为首次购物，则提示输入您所拥有购物的薪水
3. 购买商品
    直接输入商品的序号即可购买商品，并且提示剩余薪水，同时提示该商品已加入购物车
    如果薪水不足，则提示囊肿羞涩，同时提示剩余薪水
4. 查询购物清单，命令 c
    输入c可以查询本次购物清单和之前的购物清单
5. 退出，命令 q
    输入q可以退出购物，退出时会打印本次购物列表和剩余薪水

注意：用户名与密码保存在 username_passwd.txt 文件中
      锁定用户文件为 err.txt
      已购买过的用户都对用一个 username_bought.txt 文件，该文件首行保存了剩余薪水
      其他行保存了已购买的商品清单
